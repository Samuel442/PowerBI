# 🏭 Portfolio: Data Analytics & Business Intelligence for Industry

Bem-vindo ao meu repositório central de projetos de Power BI voltados exclusivamente para o setor industrial. Este espaço reúne soluções focadas em transformar dados brutos de chão de fábrica em decisões estratégicas.

## 🎯 Objetivo
Desenvolver dashboards e modelos de dados que otimizem processos, reduzam desperdícios e aumentem a confiabilidade operacional através de análise avançada de dados.

## 📊 Áreas de Aplicação nestes Projetos
- **Gestão de Energia:** Monitoramento de demanda, picos de consumo e metas de eficiência.
- **Produção e OEE:** Análise de disponibilidade, performance e qualidade por turno.
- **Sustentabilidade (ESG):** Rastreabilidade de emissões de CO2 e impacto ambiental.
- **Manutenção Preditiva:** Identificação de falhas e tendências de desgaste de equipamentos.

## 🛠️ Diferencial Técnico
Diferente de dashboards convencionais, os projetos aqui contidos são:
1. **Validados em SQL:** Todas as métricas DAX possuem queries de validação espelhadas no banco de dados (MySQL).
2. **Focados em Granularidade:** Análise de dados em tempo real ou intervalos técnicos (ex: 15 min).
3. **Escaláveis:** Modelagem de dados seguindo os padrões Star Schema para performance.
