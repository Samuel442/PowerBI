# 🏭 Power BI Industrial: Data Intelligence & Analytics

Repositório dedicado ao monitoramento e análise de eficiência energética e performance industrial. Este projeto integra dados brutos de sensores (MySQL) com dashboards estratégicos para tomada de decisão em tempo real.

## 🚀 O que este projeto resolve?
- **Monitoramento de Pico:** Identificação de picos de consumo acima do limite contratado (480 kWh).
- **Eficiência por Turno:** Comparativo de performance entre Turnos A, B e C.
- **Sustentabilidade:** Cálculo automático de pegada de carbono (Emissão de CO2).
- **Tendência de Carga:** Análise de média móvel de 7 dias para previsão de demanda energética.

## 🛠️ Tecnologias & Validação
- **Dashboard:** Power BI (DAX Avançado).
- **Banco de Dados:** MySQL (Consultas SQL para validação de métricas).
- **Confiabilidade:** 100% dos KPIs validados via SQL na View `vw_fact_consumo`.
